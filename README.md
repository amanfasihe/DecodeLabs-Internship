# Rule-Based AI Chatbot

## Project Description
This project is a simple rule-based AI chatbot developed using Python. The chatbot responds to predefined user inputs using if-else statements and runs in a continuous loop until the user exits.

## Features
- Greeting responses
- Exit commands
- User name input
- Joke command
- Continuous chat loop
- If-else decision making

## Technologies Used
- Python
- VS Code
